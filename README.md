
# Sreport - frontend
[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)


## Run Locally

Clone the project

```bash
  git clone https://github.com/Unviray/sreport-frontend
```

Go to the project directory

```bash
  cd sreport-frontend
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
